# 🧮 Advanced Calculator

A modern and responsive calculator web application developed using HTML, CSS, and JavaScript. The project provides a clean user interface and supports essential mathematical operations with a focus on usability and responsive design.

## 🔗 Live Demo

**Website:** https://vasiragini.github.io/advanced-calculator/

## 📌 Overview

This project demonstrates core front-end development concepts, including DOM manipulation, event handling, responsive layouts, and JavaScript-based calculations. It was built to strengthen practical web development skills and understand the complete deployment workflow using GitHub Pages.

## ✨ Features

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)

* Additional functionality

  * Percentage calculations
  * Clear All (AC)
  * Delete Last Character (DEL)

* User Experience

  * Responsive design
  * Clean and intuitive interface
  * Real-time display updates
  * Mobile-friendly layout

## 🛠️ Tech Stack

| Technology       | Purpose                            |

| CSS3             | Styling and responsive design      |

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/VasiRagini/advanced-calculator.git
```

### Run the Project

1. Open the project folder.
2. Open `index.html` in any modern web browser.

No additional dependencies or installations are required.
