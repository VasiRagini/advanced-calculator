# 🧮 Advanced Calculator

A responsive and interactive calculator built using **HTML, CSS, and JavaScript**.

## ✨ Features

* Basic arithmetic operations (+, -, ×, ÷)
* Percentage calculations
* Clear (AC) and Delete (DEL) functions
* Responsive design for different screen sizes
* Clean and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Live Demo

## 📂 Project Structure

```text
Calculator_project/
├── index.html
├── style.css
└── script.js
```

## 📖 How to Use

1. Open the live demo link.
2. Click the calculator buttons to enter numbers and operations.
3. Press "=" to calculate the result.
4. Use "AC" to clear the display.
5. Use "DEL" to remove the last entered character.
